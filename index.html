About Me

I am a developer from Southern California with a passion for development and healthy love of videogames. Ever since I was younger I've had a strong desire to create and innovate, and through the field of Computer Science I've been enabled to do exactly that. Over the past few years I've devoted myself to learning new skills and techniques that will allow me to become the most effective developer I can possibly be (and I'm still learning!). Recently I decided to attend a web development bootcamp at LeaningFuze to further expand and cement the knowledge I gained through college and personal studies, taking a dive into the fields of JavaScript, HTML, CSS and even SQL! With this knowledge these tools I've gained, I believe that I can be an incredible asset to any project requiring a Full Stack developer with a bit of outside knowledge.

  • Plush Stop

    [Source]
    Solo Project

    This application is a full stack Content Management System (CMS) created with React, Node.js, Express.js and PostgreSQL. The purpose behind this project is to demonstrate an ability to create and utilize a relational database that interacts with a responsive front end framework. During the initial development of this project I gained a strong appreciation for creating organized endpoints that have the capacity for conditional functionality so that the overall work on both ends is less than it otherwise would be with static function endpoints. This was the first project requiring me to implement both a front and back end for an application which posed some challenges in figuring out delegation for the system and led me to delve deeper into Postgres than I previously had before. As a result I've strengthened both my understanding of the Postgres flow and developed a greater appreciation for relational database queries and enjoyment in building them. Potential additions may include:

    • Greater amount of product details
    • Pulling details from actual vendor APIs
    • Increase of sort parameters
    • Sort parameter combination
    • Competitive price checking between vendor APIs
